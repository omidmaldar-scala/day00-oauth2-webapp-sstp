## Dependencies

- [Cats: Lightweight, modular, and extensible library for functional programming](https://typelevel.org/cats/)
- [CatsEffect: Asynchronous runtime for Scala](https://typelevel.org/cats-effect/)
- [Circe: JSON library for Scala powered by Cats](https://circe.github.io/circe/)
- [Tapir: Declarative, type-safe web endpoints library](https://tapir.softwaremill.com/)
- [Sttp: Scala HTTP Client](https://sttp.softwaremill.com/en/latest/)
- [SttpOAuth2: Oauth2 client library for Scala](https://ocadotechnology.github.io/sttp-oauth2/)
